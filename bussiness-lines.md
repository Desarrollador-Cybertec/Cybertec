# CYBERTEC — Líneas de Negocio (Estructura Oficial)

# 1. Seguridad Física

## Servicios principales

### Sistemas de Videovigilancia Avanzada

* Suministro e instalación de cámaras IP
* Grabación automática
* Detección avanzada de movimiento
* Monitoreo en tiempo real desde cualquier dispositivo

---

### Controles de Acceso Biométrico

* Diseño de soluciones según entorno del cliente
* Identificación segura mediante:

  * Tarjetas
  * Biometría
  * PIN
  * Dispositivos móviles
* Integración con:

  * Cámaras de seguridad
  * Alarmas
  * Software de gestión

---

### Sistemas de Alarmas Digitales

* Sensores inteligentes (movimiento, humo, gas, etc.)
* Alarmas audibles y sonoras
* Notificaciones en tiempo real:

  * Aplicación móvil
  * Correo electrónico
  * SMS
* Integración con sistemas de seguridad existentes
